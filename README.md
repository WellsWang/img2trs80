# bmp2trs80
通过单色BMP文件生成TRS-80 Model I Level 2 BASIC可用的绘图程序

## 目的

本程序通过解析BMP图形，利用TRS-80字符表中ASCII码128-191的图形字符，生成能够在屏幕上打印文本图像的程序。

## 程序说明

- 3lian.bmp 样例BMP文件；
- bmp2trs80.py  主程序。
  
## 使用方法
python bmp2trs80.py <BMP文件名>
